## 연료 주입

### [기능 목록]
* [x] 자동차 만들기
    - Sonata 2대, Avante 1대, K5 2대로 총 5대의 차량
    - 자동차들은 회사에 종속
* [x] 회사를 만들기
* [x] 자동차를 예약
    - 이동거리를 입력받는다.
* [x] 주입해야할 연료량을 확인할 수 있는 보고서 생성

### [예외 처리]
* [x] 자동차 이동거리 입력
  - 이동거리는 0이하를 입력받을 수 없다
  
### 차량 연비
* Sonata : 10km/리터
* Avante : 15km/리터
* K5 : 13km/리터

### [프로그래밍 요구사항]
- 상속과 추상 메소드를 활용한다.
- 위 요구사항을 if/else 절을 쓰지 않고 구현해야 한다.
- 인터페이스를 적용
